CI/CD Docker Deploy Pipeline
This project automates infrastructure provisioning and application deployment using Terraform, GitHub Actions, and Ansible.
Overview
Terraform provisions AWS EC2 infrastructure with security groups.
GitHub Actions builds and pushes Docker images to Docker Hub.
Deployment occurs via SSH triggered by GitHub Actions or manually via Ansible.
Prerequisites
AWS Account with CLI credentials
Docker Hub Account
GitHub Repository
EC2 Key Pair (devops-key.pem)

Setup Instructions
Provision Infrastructure
terraform init
terraform apply
Update the EC2_HOST secret with the public IP generated by Terraform.
Trigger Deployment
Push code to the main branch to automatically build and deploy via GitHub Actions.
Alternative Ansible Deployment
To deploy manually using Ansible instead of GitHub Actions:
ansible-playbook -i inventory.ini playbook.yml
